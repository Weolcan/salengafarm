# Salenga Farm Inventory System

This is a Laravel-based inventory system designed for **Salenga Farm** in Digos, Davao del Sur.

## Project Overview
The system helps manage plant inventories, automate misting for plants, and monitor farm operations efficiently.

### Features
- Plant inventory tracking
- Automated misting system control
- User management for farm staff
- Inventory reports and analytics
- Subscription-based notifications (if implemented)
