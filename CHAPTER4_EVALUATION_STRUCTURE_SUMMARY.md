## Evaluation Results Structure (Based on Samples)

### Section 1: Introduction to Evaluation Results
- Brief paragraph explaining the purpose of evaluation
- Mention ISO 9126/25010 framework
- State who the respondents are (total count)

### Section 2: Functional Suitability Tables (By User Role)

**Sample 1 Format:**
- Separate tables for each user role
- Table structure:
  - Column 1: Assessment Description (specific features)
  - Column 2: Mean (numerical rating)
  - Column 3: Verbal Interpretation (Strongly Agree, Agree, etc.)
  - Last row: Consolidated Mean
- Paragraph BEFORE table: Introduces the table and overall rating
- Paragraph AFTER table: Discusses highest/lowest rated features with citations

**Sample 2 Format:**
- Similar structure but simpler
- Focuses on key features only
- Shorter descriptions
